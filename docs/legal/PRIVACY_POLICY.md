# Quant 交易台（QuantTradingApp）隐私政策

生效日期：v1.0.0 发布前由运营主体填写
最近修订：2026-05-16
适用版本：Quant 交易台 Android v1.0.0（versionCode 1）及以上

本隐私政策说明Quant 交易台应用（以下简称"本应用"或"我们"）在 Android 客户端及配套后端服务中收集、使用、存储和共享个人信息的方式。请在使用本应用前完整阅读本政策。继续使用即视为您已阅读并同意本政策。

本政策遵循《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《中华人民共和国数据安全法》《App 违法违规收集使用个人信息行为认定方法》《常见类型移动互联网应用程序必要个人信息范围规定》以及国家网信办和工信部相关合规要求。

## 1. 我们处理的个人信息

### 1.1 账号与登录信息

- 注册时收集：昵称（默认"本机用户"）、手机号（11 位）、密码（仅以 PBKDF2-SHA256 + per-user salt、120,000 次迭代的哈希形式存储，120,000 次迭代不可逆）、设备标识（用户安装应用时由系统提供）。
- 登录与会话：访问令牌（access token，有效期 2 小时）、刷新令牌（refresh token）、会话所属设备标识、会话过期时间。
- 法律依据：履行您与我们之间的服务合同（个人信息保护法第十三条第二项）。

### 1.2 订阅与订单信息

- 订单：订单 ID、订阅档位（选股 VIP / 量化 VIP / 全功能 VIP）、订阅时长、金额（分）、币种（CNY）、支付渠道（WECHAT / ALIPAY）、客户端订单 ID、设备标识、订单状态（PENDING / PAID / REFUNDED / CANCELLED）、创建时间、支付时间。
- 支付回调审计：支付回调记录、第三方流水号、回调金额、是否接受、详情、创建时间。
- 权益记录：选股 VIP 到期时间、量化 VIP 到期时间、宽限期截止时间、权益变更来源（注册 / 支付订单 / 退款 / 取消）、更新时间。
- 法律依据：履行付费订阅合同、支付结算法律义务、税务和审计留存义务。

### 1.3 研究行为数据（本机存储为主）

- 自选池、筛选条件、自定义量化策略、复盘快照、市场宽度记录、研究笔记。
- 本机社区帖子和评论草稿（当前版本社区为本机演示，不上传至公共社区）。
- 通知偏好：每日研究提醒是否开启。
- 上述数据当前主要保存在本机 Room 数据库；正式版本若引入云端同步会另行更新本政策并提示用户。
- 法律依据：履行您与我们之间的服务合同。

### 1.4 行情与 K 线数据缓存

- 最近一次成功获取的股票 quote 样本、数据源标识、获取时间、缓存有效区间。
- 该缓存用于离线展示，不包含个人身份字段。

### 1.5 设备与日志信息

- 设备标识、网络请求路径与状态、请求时延、响应时间戳（用于后端结构化日志与速率限制）。
- 操作系统报告的崩溃栈与匿名错误码（如启用应用商店或第三方崩溃统计渠道，会在本政策第 6 节披露）。
- 法律依据：保障应用运行与服务安全的合法利益（个人信息保护法第十三条第三项）。

### 1.6 我们不收集的信息

- 我们不收集您的精确地理位置、通讯录、相册、照片、短信、通话记录、剪贴板、麦克风或摄像头数据。
- 我们不在客户端调用任何 IMEI / MEID / IMSI / SIM 卡序列号 / Android ID 等敏感设备识别符的隐私接口。
- 我们不读取您手机内的其他应用数据。
- 我们不接入券商交易通道，因此不收集您的资金账户、银行卡明细、券商账号或交易记录。

## 2. 我们如何使用信息

- 完成账号注册、登录、会话刷新、账号删除。
- 展示本机和服务端 VIP 权益状态、订单状态、支付凭证与退款/取消状态。
- 保存与展示自选池、复盘、量化策略、研究纪要、市场快照。
- 在网络不可用时回退到本机行情缓存。
- 发送每日本地研究提醒（仅在您开启通知权限后）。
- 校验支付回调、订单签名、权益续期、退款与取消，必要时配合反欺诈与审计核查。
- 改进应用稳定性、安全性、性能、可用性。

我们不会基于上述信息进行用户画像广告、第三方营销分发或自动化决策。

## 3. 设备权限

| 权限 | 用途 | 是否必需 |
| --- | --- | --- |
| INTERNET（网络） | 获取公开行情、指数、K 线，以及账号、订单、权益、退款的后端 API | 必需 |
| ACCESS_NETWORK_STATE（网络状态） | 判断离线/在线，决定是否走本机缓存 | 必需 |
| POST_NOTIFICATIONS（通知） | Android 13+ 上推送每日研究提醒 | 可选，可随时关闭 |
| RECEIVE_BOOT_COMPLETED（开机广播） | 设备重启后恢复每日本地提醒 | 可选 |

拒绝可选权限不影响核心研究功能的使用。

## 4. 数据存储与跨境传输

- 后端默认部署区域：中国大陆境内云服务商节点；具体部署区域以应用商店上架信息和运营主体页面披露为准。
- 本应用当前不进行跨境数据传输。如未来需要跨境传输，我们会按《个人信息出境标准合同办法》或《数据出境安全评估办法》履行评估、备案或合同义务后再行实施，并通过应用内或公告页面通知您。
- 后端使用 SQLite 持久化，启用结构化 JSON 日志和路径级速率限制。生产环境使用 HTTPS。

## 5. 数据保存期限与删除

- 账号、订单、回调审计：在账号存续期内按合同与监管要求保留。
- 支付凭证、退款凭证、商户回调：依据国家税务、反洗钱与支付结算相关法规留存至少 5 年。
- 本机研究记录：保留在本机 Room 数据库，可通过卸载应用或在应用内"账号 > 注销账号"删除。
- 账号注销：在"账号"页点击"注销账号"后，应用调用 `DELETE /v1/me`，服务端删除账号、会话、权益与订单回调审计明细（保留法律要求的最小化匿名审计聚合），同时本机清理用户状态、自选池、持仓、帖子、评论、策略、复盘快照、行情缓存。
- 您也可以通过本政策第 11 节列出的联系方式行使删除请求权。

## 6. 第三方数据源与处理者

| 类别 | 用途 | 个人信息共享 |
| --- | --- | --- |
| 公开行情接口（新浪、东方财富等公共接口） | 获取行情、K 线、指数样本，仅用于研究展示 | 不共享个人信息；请求中只携带股票代码等公共字段 |
| 微信支付 / 支付宝（生产版） | 处理订阅订单支付与回调 | 仅传递订单号、金额、订阅档位；不传递昵称或密码 |
| 应用商店分发渠道（华为应用市场 / 小米应用商店 / 应用宝 / Google Play 等） | 渠道分发、版本更新、崩溃统计（按各渠道默认策略） | 各渠道自身收集设备与崩溃信息，详见对应商店隐私政策 |
| 授权数据提供商 | 主力资金流、龙虎榜、财务指标、估值百分位等高级字段 | 当前 MVP 未接入；正式接入前会在本节披露具体提供商、数据类型与隐私链接 |

我们不会主动向上述任何第三方共享您的密码、手机号验证记录、身份证号或其他敏感个人信息。

## 7. 您的权利

依据《个人信息保护法》第四章，您享有以下权利：

- 知情权与决定权：阅读本政策，决定是否继续使用本应用。
- 查阅权与复制权：通过"账号"页查看个人信息、订单与权益记录，并可联系客服申请复制副本。
- 更正权：通过"账号"页修改昵称等可变信息，或联系客服更正。
- 删除权：在"账号 > 注销账号"中删除账号；也可联系客服。
- 撤回同意权：撤回对通知、可选权限或可选数据收集的同意。
- 转移权：在合理且技术可行范围内，请求将个人信息转移到您指定的接收方。
- 投诉权：向我们投诉处理结果；如对答复不满意，可向网信、市场监管或公安等监管机关投诉。

我们将在 15 个工作日内回复您的请求。

## 8. 未成年人保护

本应用面向具备独立投资研究能力的成年人，不面向未满 18 周岁的未成年人提供服务。我们不会在明知用户为未成年人的情况下处理其个人信息。监护人发现未成年人误用本应用并产生订阅或个人信息留存的，可联系客服删除相关数据并申请退款（订阅渠道的退款规则以应用商店或商户平台规则为准）。

## 9. 信息安全

- 密码使用 PBKDF2-SHA256 + per-user salt、120,000 次迭代单向哈希。
- 访问令牌、刷新令牌随机生成且与设备绑定。
- 支付回调使用 HMAC-SHA256 与 5 分钟时间窗签名校验，启用幂等去重。
- 后端管理审计页（/admin）需要管理员令牌，仅运营主体内部使用。
- 客户端不存储生产支付回调密钥、后端管理令牌、授权数据源凭据。
- 我们启用代码混淆（R8）和资源裁剪降低逆向风险，但本机 VIP 状态非防篡改，正式支付以服务端为准。

## 10. 政策更新

我们可能根据法律法规、监管要求或产品调整更新本政策。更新后会在应用启动页或"账号 > 隐私政策"页提示，并保留历史版本备查。重大变更（如新增数据类别、新增对外共享对象、跨境传输等）会在生效前以显著方式通知用户。
