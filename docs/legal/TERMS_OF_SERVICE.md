# Quant 交易台用户协议

最近修订：2026-07-05

本协议适用于 Quant 交易台 Android 客户端的公开代码版本。当前项目是本机研究演示工具，不包含券商交易或实盘下单功能。

## 1. 服务定位

- 本应用提供股票研究记录、行情样本观察、复盘、历史模拟、自定义策略管理和本机社区研究记录工具。
- 本应用不接入券商交易通道，不提供下单、跟单、代客理财、资金托管或具体买卖指令。
- 本应用不构成证券投资咨询服务、证券投资顾问业务或基金销售业务。
- 本应用展示的评分、风险雷达、模型诊断、研究纪要、市场宽度指标和历史模拟均基于公开样本或本机数据，不能代表未来收益。

## 2. 本机账号与数据

- 账号信息只用于本机演示登录状态，默认保存在设备本地 Room 数据库。
- 你可以在账号页删除本机账号与研究资料。
- 删除操作会清理本机账号状态、自选池、目标价提醒、持仓、帖子、评论、策略、复盘快照和行情缓存。
- 本公开版本没有云端账号系统或同步系统。

## 3. 用户行为规范

你承诺不会通过本应用实施以下行为：

- 发布明确的买入、卖出、加仓、减仓、空仓等交易指令。
- 发布“保证收益”“稳赚”“必涨”“翻倍”“跟单回报”等夸大或虚假承诺。
- 代客理财、资金托管、非法荐股、有偿荐股或导流到违法、灰色业务。
- 抓取、复制、分发超出源站授权范围的数据，或对本应用进行恶意逆向、修改、二次分发。
- 攻击、干扰第三方数据源、设备或网络服务。

## 4. 数据与免责声明

- 行情和 K 线数据来源于公开接口或本机缓存，可能存在延迟、缺失、错误、字段调整或源站不可用。
- 所有评分、诊断、研究纪要、模型信号、风险标签和历史模拟结果均为研究工具输出，不构成投资建议或具体买卖指令。
- 历史模拟结果受样本区间、数据源质量、费用假设、滑点假设和模型参数影响。任何历史表现均不能代表未来收益。
- 你应独立判断投资风险，自行承担投资决策后果。

## 5. 知识产权与开源使用

- 仓库代码按仓库声明的许可证和 GitHub 展示状态使用。
- 你在应用内创建的自选池、策略、笔记等本机内容属于你自己。
- 引用或复用本项目时，请保留必要的来源说明，并遵守第三方依赖和公开数据源的使用条款。

## 6. 协议更新

公开仓库可能随代码整理、依赖升级或文档修订而更新本协议。继续使用即视为接受更新后的说明。
